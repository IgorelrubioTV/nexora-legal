Nexora Privacy Policy

How Nexora stores and uses data required to operate the bot.
Last updated: March 14, 2026

1. Data We May Store

Nexora may store limited Discord-related data required for bot functionality, such as server IDs, channel IDs, role IDs, user IDs, ticket data, moderation data, and server configuration settings.

2. Purpose of Data

This data is used only to operate bot features such as moderation, tickets, logging, leveling, automations, and configuration saving.

3. Data Sharing

Nexora does not sell personal data. Data is not shared except when required by law or necessary for infrastructure providers that keep the service running.

4. Data Removal

Server administrators may remove Nexora from a server at any time. Reasonable requests for data deletion may be honored when applicable.

5. Security

We take reasonable measures to protect stored data, but no method of storage or transmission is guaranteed to be completely secure.

6. Contact

For privacy questions, contact the Nexora developer through the official support server or listed contact methods.
